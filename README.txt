Para que o programa funcione corretamente, se recomenda a versão 18.0.2 do Java

Possui mais de uma política de substituição.

1 - Pasta Util
a)CliParser
    Conversão de entrada e geração da cache
b)FileManager
    Manipula os arquivos
c)RuntimeData
    Conta hits e misses. Também imprimi uma string
    de output com o formato flag_saida escolhido
d)Util
    Contém métodos de utilidade como conversores e geração
    de números pseudoaleatórios

2 - Pasta Memoria
a)Bloco
    Contém as linhas e conjuntos da cache
b)Cache
    Classe principal da cache
